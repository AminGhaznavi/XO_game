# Tic-Tac-Toe Game 🎮

## 📖 Description
This project is a simple **Tic-Tac-Toe** game implemented in C.  
The game runs on a 3×3 board where two players take turns using symbols `X` and `O`.

## ⚙️ How to Run
Compile and run the program using GCC:

```bash
gcc main.c -o tictactoe
./tictactoe
```
🗂️ Project Structure
main.c → Contains all game logic (structures, functions, and main loop).

✨ Features
Displays the game board in the terminal.

Checks for winning conditions after each move.

Detects draw when the board is full.

Allows replay after each round.

Automatically manages player turns.
